# Authenticated To-Do List Web App (MERN)

A powerful and intuitive **MERN stack To-Do application** designed for productivity and simplicity. With secure user authentication and a clean, mobile-first interface, this app helps users organize tasks across different timeframes — daily, weekly, monthly, and yearly — while automatically managing task lifecycles.

## Features
- **User Authentication** – Secure login and signup with JWT-based authentication.
- **Smart Task Management**  
  - Create, edit, and delete tasks.  
  - Categorize tasks as **Daily, Weekly, Monthly, or Yearly**.  
  - Automatic deletion based on type:  
    - Daily tasks expire after 24 hours  
    - Weekly tasks expire after 7 days  
    - Monthly tasks expire after 30 days  
    - Yearly tasks expire after 365 days  
- **Perfect Mobile Responsiveness** – Optimized for all screen sizes with a mobile-first design.
- **Simple & User-Friendly UI** – Clean layout focused on usability and productivity.
- **MERN Stack** – Built with MongoDB, Express.js, React, and Node.js for a robust full-stack solution.

## Tech Stack
- **Frontend:** React + Tailwind CSS  
- **Backend:** Node.js + Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Tokens)
